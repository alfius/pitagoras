Pitagoras
=========

Overview
--------

A very basic Paint application for Windows, built with the Windows API. This project intends to be merely demonstrative and not a complete graphic design solution.
Language: Spanish

Pit√°goras (spanish for Pythagoras) is a very basic Paint application intending to show basic usage of the Windows API in C++. It offers basic functionality like creating, opening, editing and saving a bitmap. The edition tools are restricted to:

* Pen
* Rectangle
* Ellipse
* Fill
* Choose line and fill color
